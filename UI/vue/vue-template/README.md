# vue-template

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```

###=== 1 ===
使用vue-cli快速搭建

安装 :
npm install -g @vue/cli
#OR
yarn global add @vue/cli

创建一个项目：
vue create vue-project
#OR
vue ui

运行项目：
yarn serve
yarn run serve
npm run serve

构建项目：
yarn build
